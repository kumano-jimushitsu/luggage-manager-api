# Luggage-api

Go API for Kumano luggage management app.

# Version
```
go1.17.2
```

# Build

```
cd server
go mod download  // download modules
```

# Run
```
cd server
go build -o . ./main/server.go // bundle go files
./server  // execute
```

the host is localhost:8080
